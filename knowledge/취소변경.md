# 예약 취소 및 변경 안내

## 취소 방법
1. **챗봇**: ③번 메뉴(취소/변경) 선택 → 확인번호 입력 → 취소 완료
2. **포털 마이페이지**: 로그인 → 예약 내역 → 취소 버튼 클릭
3. **전화**: 관리사무소 031-470-0242 (운영시간 내)

## 취소 가능 기한
- 방문 전날(D-1) 오후 6시(18:00)까지 취소 가능
- 기한 이후 취소 불가
- 미취소 상태로 미방문 시 당일 미입차로 처리되어 제재 적용

## 환불
- 취소 기한 내 취소 시 전액 환불
- 환불 소요: 3~5 영업일
- 원래 결제 수단으로 환불

## 날짜 변경
- 직접 날짜 변경 기능은 없습니다.
- 기존 예약 취소 → 새 날짜로 재예약 해주세요.
- 변경하려면 취소 기한(D-1 18:00) 전에 처리해야 합니다.

## 차량번호 변경
- 차량번호 변경이 필요한 경우 기존 예약 취소 후 새 차량번호로 재예약해 주세요.
- 또는 관리사무소(031-470-0242)에 전화하시면 현장에서 변경 가능합니다.

## 확인번호 분실
- 예약 시 발송된 SMS에서 확인번호를 확인할 수 있습니다.
- SMS도 분실한 경우 관리사무소(031-470-0242)에 차량번호와 예약일을 알려주시면 조회해드립니다.
