# 주차장 예약 방법

## 예약 절차 (6단계)
1. 챗봇 ②번 메뉴(주차 예약) 선택 또는 "예약하고 싶어요" 입력
2. 달력에서 원하는 날짜 선택 (오늘~7일 후까지 가능)
3. 잔여 면수 확인
4. 차량번호 입력 (예: 12가3456)
5. 할인 대상 선택 (해당 시)
6. 예약 정보 확인 → [예약 확정] 클릭 → 결제 → 확인번호 발급

## 예약 규칙
- 예약 가능 기간: 오늘부터 7일 후(D+7)까지
- 월 1회 예약 제한: 1인(1차량) 기준 월 1회만 예약 가능
- 만차(잔여 0면)인 날짜는 예약 불가
- 휴원일(매주 월요일, 공휴일)은 예약 불가

## 총 주차면
- 50면 (전면 사전예약제)
- 현장 주차 불가, 반드시 사전예약 필요

## 예약 확인
- 예약 완료 시 확인번호(AY로 시작)가 발급됩니다.
- 확인 SMS가 등록된 휴대폰으로 발송됩니다.
- 입차 시 차량번호 자동 인식으로 입장합니다.

## 주의사항
- 예약한 차량번호와 실제 입차 차량이 다르면 입장이 거부됩니다.
- 예약일에만 유효하며, 다른 날짜에 사용할 수 없습니다.
- 월 예약 횟수를 초과하면 예약이 제한됩니다.
